# ITtalk
ITtalk-모험하는멍게
MVC패턴으로 구현한 Web Application

### 주제
개발자와 개발자를 꿈꾸는 사람들의 커뮤니티 웹 서비스

### 기획 의도
IT업계에 종사하기를 희망하는 분 혹은 이미 재직중인 분들이 하나의 커뮤니티 사이트에 모여 필요한 지식을 얻고, 서로 돕고 배울 수 있는 소통의 창구를 만들고 싶었습니다.  이를 계기로 “ITtalk(잇톡)” IT개발자 커뮤니티 사이트를 기획하였습니다.

### 역할 분담
Model : 정준, 김민석

View : 황명연

Controller : 황명연, 박정인

### 사용언어
HTML, JSP, JAVA
